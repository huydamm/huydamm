### 🔭 What I'm into

- 🤖 **Applied AI / Computer Vision** — training models to read cards, faces, and emotions, then shipping them somewhere useful
- 🌐 **Full-stack web** — FastAPI / Flask backends with React frontends, deployed and live
- 🔌 **Hardware + code** — Arduino and Raspberry Pi projects where the software meets a physical sensor

### 🚀 Featured projects

| Project | What it does |
|---------|--------------|
| [**UmassQuite**](https://github.com/huydamm/UmassQuite) | Crowdsourced study-spot finder for UMass — live quiet scores + an ML model that *predicts* where it'll be quiet next |
| [**CardCam**](https://github.com/huydamm/CardCam) | Poker equity calculator that reads real cards from a camera (OpenCV + CNN) and runs an exact Texas Hold'em engine |
| [**MoodSync**](https://github.com/huydamm/MoodSync) | Reads your face *and* your words at once to detect your mood in real time (UMass hackathon project) |
| [**PokemonVGC-Calculator**](https://github.com/huydamm/PokemonVGC-Calculator) | Fast Gen 9 damage calculator that auto-fills the opponent's most common set |
